How to get your Client ID and Secret
  1. Go to your Shopify Admin → Settings → Apps and sales channels
  2. Click "Develop apps" at the top right
  3. Create a new app, then "Configure Admin API scopes"
  4. Enable: read_themes, read_script_tags, read_pixels, read_custom_pixels, read_customer_events
  5. Save, then "Install app"
  6. Under "API credentials" tab → copy the Client ID and Client Secret
